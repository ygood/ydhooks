# useTitle

设置页面title

## 代码演示

### 基础用法

<code hideActions='["CSB"]' src="./demo/demo1.tsx"></code>

## API

```typescript
useTitle(string, reset);
```

### 参数

| 参数   | 说明                                 | 类型    | 默认值 |
| ------ | ------------------------------------ | ------- | ------ |
| string | 需要设置的title                      | string  | -      |
| reset  | 销毁时是否需要将title变回之前的title | boolean | false  |
